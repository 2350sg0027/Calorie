# カロリー計算ツール

## 概要

商品ごとの価格・カロリー・塩分量を管理し、個数に応じた合計値を算出できるWebアプリケーションです。ユーザーが直感的に操作できるよう、シンプルなUIと分かりやすい表示を意識して設計しました。

## デモ

[https2350sg0027.github.ioCalorie](https://2350sg0027.github.io/Calorie/)

## 主な機能

- 商品一覧の表示
- 個数の増減（＋  − ボタン）
- 合計金額・合計カロリー・合計塩分の自動計算
- 商品の追加・削除機能
- localStorageによるデータ保存

## 工夫点

- localStorageを活用し、ユーザーの操作状態を保持できるようにした
- 小数計算の誤差に対して丸め処理を行い、表示の正確性を向上させた
- UIをシンプルに設計し、直感的に操作できるよう工夫した

## 開発プロセス

1.要件定義

カロリーや価格を簡単に管理できるツールを作成することを目的とし、必要な機能（表示・計算・追加・削除）を整理した。

2.データ設計

商品情報（名前・価格・カロリー・塩分）をJSON形式で管理し、拡張しやすい構造とした。

3.基本機能の実装

JavaScriptのforEachメソッドを用いて配列データを処理し、合計値の計算を実装した。

4.機能拡張

localStorageを導入し、追加・削除したデータがブラウザ上に保存されるようにした。

5.UI改善

CSSを用いてカード形式のレイアウトを採用し、視認性と操作性の向上を図った。

6.動作確認

Live Serverおよびブラウザ上で動作確認を行い、計算結果や表示に不具合がないか検証した。
